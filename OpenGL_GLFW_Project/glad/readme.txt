The GLAD library comes with 2 options when used in a C/C++ project. The difference between the two is that 
one is meant for debugging while the other is not. To be able to support both versions (with the desired version used
based off the project's build settings), some modifcations have been made to the filesystem layouts of each version,
which then necessitated that several of the '#include' statements in the source code be modified so that the proper
files are being found. Besides from changes to the '#include' statements, there have been no modifications made to
any of the GLAD files from the versions generated by the following links:
